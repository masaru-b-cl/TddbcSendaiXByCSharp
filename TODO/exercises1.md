いちごを出荷するには、品種とサイズを知る必要があります。


- [ ] 品種とサイズを与えて、いちごを作成する
    - [ ] いちごの品種
        - [ ] あまおう
        - [ ] とちおとめ
        - [ ] もういっこ
    - [ ] いちごのサイズ
        - [ ] S
        - [ ] M
        - [ ] L
        - [ ] LL
- [ ] いちごを文字列表現に変換する
    - [ ] "<品種>: <サイズ>"形式になる
        - [ ] 「あまおう」の「S」サイズは"あまおう: S"になる
